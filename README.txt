A local, low-pressure AI chatbot designed to support individuals experiencing social withdrawal (hikikomori)

This project is designed for local use. Users MUST install Ollama to enable AI responses and Node.js to run JavaScript. Upon running start.bat, it will redirect the user to the website for the installation of Ollama. Alternitively, the user can have it pre-installed and it will work just fine. USERS NEED TO INSTALL NODE.JS. The links are listed below:
- Ollama: https://ollama.com/
- Node.js: https://nodejs.org/en/download

PLEASE RESET COMPUTER/LAPTOP AFTER INSTALLATION

Note: THIS IS MERELY A SIMULATION OF THE WEBSITE IF IT WERE TO GO PUBLIC. This website and service is accessible locally only
